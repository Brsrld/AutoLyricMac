# AutoLyricMac Visual Style Guide

Inspect the actual reference files when available. These rules are the intended baseline.

## Archive Collage (`archiveCollage`)

- 1080x1920 warm off-white paper canvas with large intentional negative space.
- Monochrome archival photos appear as movable framed objects, not permanent full-screen backgrounds.
- Vary position and size (roughly 55–90% width, 25–60% height); use subtle shadows and layered translucent grey/black/white rectangles.
- Softened contrast, lifted blacks, controlled grain, occasional dust/scratches/flicker, mild blur; avoid glossy HDR, neon, fast zooms, and heavy glitches.
- Main scenes follow lyric phrases: calm 4–7s, normal 3–5s, energetic 2–4s. Beats only affect micro-motion.
- Motion: slow push/pull, gentle drift, rotation below about 1.5°, and slow layer repositioning.
- Transitions: soft crossfade, fade through white/near-black, block wipe, layered dissolve; typically 0.25–0.8s.
- Subtitles: original line and optional Turkish translation on separate irregular cream paper cutouts; dark editorial serif/typewriter text; slight rotation/shadow; dynamic central or lower-middle placement without covering faces.
- The vertical layout must be redesigned as a tall artboard, not a square video centered with borders.

## Doodle Memory (`doodleMemory`)

- 1080x1920 full-frame warm nostalgic imagery: amber/yellow highlights, muted olive/green shadows, faded brown, mild analog grain and soft contrast.
- Background subjects: home, kitchen, childhood, windows, parks, streets, family memories, sunlight, nature, ordinary life.
- Transparent doodles use white/cream fill, dark navy imperfect hand-drawn outlines, simplified rounded forms, minimal facial detail, and consistent line width.
- Doodles must interact with the real environment (beside a window, on a bench, near a sink/stove), not float as arbitrary stickers.
- Each meaningful lyric phrase maps to a visual metaphor and normally a new scene; typical scenes 2.5–5.5s.
- Background motion is subtle; doodles may slide in, breathe 1–2%, rotate slightly, or bounce minimally on emphasis.
- Transitions: clean phrase cut, very short dissolve, paper wipe, or sticker pop; 0–0.35s. Avoid long crossfades.
- Handwritten dark-navy lyrics use irregular white sticker/paint borders and dynamic placement around the subject/doodle, never one fixed bottom box.
- Start with a curated semantic doodle library and normalize alpha, outline, stroke, and fill. No gibberish text or watermarks.

## Automatic selection

- Prefer Archive Collage for folk/old/historical, melancholic, travel, railway, separation, or documentary tone.
- Prefer Doodle Memory for warm, romantic, childhood, domestic, intimate, playful, memory-driven songs and short visual metaphors.
- If confidence is low, default to Archive Collage. Explicit selection always wins.

## Shared rules

Both styles must match scenes to lyric meaning, change main scenes mainly at phrase boundaries, use licensed high-resolution media, preserve safe readable subtitles, avoid stretching/watermarks, remain coherent, feel handmade/analog, and never look like a generic automatic slideshow.
